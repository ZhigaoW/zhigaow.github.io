# 鲜衣怒马少年时
最近看了华哥的课程，结合自己的思考，有一些感悟。

我常常幻想自己做一件事情如何成功，非常开心，但是回到现实中来又感觉差距非常大；同时为了不失去信心，我又会拿一些名人的事例鼓励自己，表示自己离那些大佬也不是那么远；反反复复就造成了自己对自己情况判断的不清晰，不知道自己该做什么事情才能变得更好，甚至非常焦虑的同时也没有目标。

其实做成一件事情的道理就是专项训练加上时间的积累，二者联合起来是一个建立高质量心理表征的过程。

目前对于我来讲商业上的专项训练就是模仿一门生意和不停的深入的拆解生意。

时间上的积累这一点非常微妙，它说明了一个道理就是其实最终的成功并不是取决于智商，是坚持，或者叫百折不挠。当然如果能坚持下来说明有内在驱动力，华哥说驱动力很简单，就是强化坚持的理由，弱化放弃的理由。

坚持的理由

1. 这个世界上大部分人是平庸的人，我不想当平庸的人。
2. 穷是非常可怕，它会让你没有时间感受这个世界的很多美好的事情。
3. 放弃的人没有成功
4. 建立高质量的心理表征



可能放弃的理由
1. 我没有这个天赋（成功与天赋无关）
2. 这件事情会花很多钱（通过其他的渠道拿钱）
3. 这件事情对我来讲具有巨大的不确定性
4. （多做撕裂自己的专项训练，提高自己增加确定性）
5. （人生只活一次，必须要莽一点）
6. ...
