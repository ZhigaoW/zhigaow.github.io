红黑树

性质

  1. 根结点为黑色
  2. 外部(叶子)结点为黑色
  3. 红色结点的子结点为黑色 => 红色结点的父结点和子结点都是黑的
  4. 外部(叶子)结点到根结点，途中黑色结点的数目相同 => 限制黑色的深度


      5. 将每一个红结点提升到父亲结点，将得到一颗(2, 4)-B树

      用途

      基本算法

      插入

      插入一个结点，首先其满足BST的性质，其次为了满足RBTREE的更多性质，我们将其染为红色
      染为红色的结点插入满足性质1、2、4，但是有可能破坏性质3，我们称为双红缺陷
